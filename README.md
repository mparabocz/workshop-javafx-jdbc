# Workshop JavaFX JDBC

This project was developed as part of my studies in Java Desktop Applications and Database Connectivity.  
It demonstrates a complete CRUD system with a graphical interface using JavaFX, integrated with a MySQL database via JDBC, following a layered architecture approach.

## Project Goals

- Build a desktop application using JavaFX
- Integrate a relational database using JDBC
- Implement full CRUD operations with UI interaction
- Apply layered architecture (GUI + Service + DAO)
- Handle validation and user input errors
- Manage database exceptions and constraints

## Technologies Used

- Java 25
- JavaFX
- JDBC
- MySQL
- Git & GitHub

## Features

- Graphical user interface with JavaFX
- Create, Read, Update and Delete records, with integrity validation
- Form validation with user feedback

## How to Run

1. Create a MySQL database as described in `database.sql` (or equivalent)
2. Copy `db.properties.example` to `db.properties`
3. Configure your database credentials in `db.properties`
4. Run the application

## Concepts Practiced

- JavaFX (FXML, Controllers, UI components)
- JDBC API
- Layered architecture (DAO, Service, GUI)
- MVC pattern adaptation
- Exception handling (custom and SQL)
- Input validation and error handling
- Clean code practices
